# 📦 Backend Training Assignments

## 📘 Introduction

This repository contains a collection of backend training assignments that I’ve been working on to explore and practice **Node.js** and backend development. These tasks have helped me get familiar with core concepts such as logic building, working with modules, handling files, and using the command line.

### 📁 Structure & Purpose
Each assignment is self-contained and focuses on a specific backend concept. The folders are organized to help me (and others reviewing the work) easily navigate through different topics and solutions.

The project structure also helps separate backend-focused logic from frontend components, especially as the assignments grow in complexity.
