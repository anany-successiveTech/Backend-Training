# TypeScript Guide

## What is TypeScript?

TypeScript is a **strongly typed** programming language that builds on JavaScript by adding **static type definitions**. It is developed and maintained by Microsoft.

### Key Features:
- Static typing
- Type inference
- Interfaces and Type Aliases
- Enhanced tooling with IDEs (like VS Code)
- Compiles down to clean JavaScript
