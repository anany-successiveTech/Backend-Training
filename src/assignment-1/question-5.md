# Step 5: Set Up Nodemon for Auto-Restarting the Server

`nodemon` is a helpful tool that automatically restarts your Node.js application whenever you make changes to your source files. This saves you from having to manually stop and restart the server every time you update your code.

## Why Use Nodemon?

- Automatically watches for file changes
- Instantly restarts your server after saving changes
- Speeds up the development workflow

## Installation

To install `nodemon` globally on your system, run the following command in your terminal:

```bash
npm install -g nodemon
