# TypeScript Basics

This document provides a concise overview of TypeScript — a typed superset of JavaScript that compiles to plain JavaScript.

##  What is TypeScript?

TypeScript is a strongly typed, object-oriented, compiled language. It builds on JavaScript by adding static type definitions.

##  Benefits of TypeScript

- Static typing for better tooling and error checking
- Enhanced code readability and maintainability
- Powerful tooling with autocompletion and type inference
- Early detection of bugs
